## ezAdmin

### Convert-AdSidToHex.ps1

Converts an Active Directory SID to hexadecimal format.

```powershell
.\Convert-AdSidToHex "S-1-5-21-3623811015-3361044348-30300820-1013"
```

Credits to [Raymond Chen](https://github.com/oldnewthing) for the explanation<sup>[1](https://devblogs.microsoft.com/oldnewthing/20040315-00/?p=40253)</sup> that made the *Convert-AdSidToHex.ps1* script possible.
